# homebrew-tap

Homebrew tap for [@reidransom](https://github.com/reidransom)'s tools.

## Usage

```bash
brew tap reidransom/tap
brew install gojekyll
```

`brew install gojekyll` pulls in [`dart-sass`](https://formulae.brew.sh/formula/dart-sass)
automatically, so SCSS/Sass support works out of the box.

## Formulae

| Formula | Description |
|---|---|
| [`gojekyll`](https://github.com/reidransom/gojekyll) | Go clone of the Jekyll static site generator |

Formulae in this tap are generated and published automatically by
[GoReleaser](https://goreleaser.com) when a new version is tagged in the source
repository.
